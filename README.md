# Portfolio Demo Site (v1)

HTML, CSS ve JavaScript öğrenme sürecimde geliştirdiğim ilk portfolio demo sayfası.

## Özellikler
- Hero / Ana sayfa
- Hakkımda
- Yetenekler (progress bar)
- Projeler
- İletişim (demo form)

## Teknolojiler
- HTML5
- CSS3
- JavaScript (DOM / Events)

## Not
Bu proje demo amaçlıdır ve zamanla geliştirilerek portföye dönüştürülecektir.
